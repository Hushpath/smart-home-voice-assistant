| FR-013 | 天气查询 | 已实现/模拟回退 | Open-Meteo 优先，失败回退本地模拟数据 |
| FR-014 | 场景模式 | 已实现 | 初始化回家、睡眠、离家模式，作用于虚拟设备 |
| FR-015 | 个性化偏好 | 已实现 | 默认方言、默认输入方式、学习建议 |
| FR-016 | 设备别名 | 已实现 | 当前用户隔离，指令可命中别名 |

## 13 当前已知限制

1. 系统为课程原型，设备为 SQLite 虚拟设备，不接入真实智能家居硬件。
2. 语音识别不训练自有模型；云端 ASR 需要后端配置密钥和网络。
3. `MockASRProvider` 只用于 pytest、smoke test 或本地开发测试。
4. 浏览器 Web Speech API 由前端调用，依赖浏览器支持。
5. 规则式 `CommandParser`、`DialectNormalizer` 和 `MultiCommandParser` 适配课程演示常见表达，不是通用自然语言理解系统。
6. 提醒模块只做数据管理，不含后台定时任务、通知推送或系统级闹钟。
7. 天气查询可访问 Open-Meteo 时使用真实数据，失败时回退本地备用数据，测试中禁用真实天气请求。
8. 当前未执行压力测试、安全渗透测试、多浏览器兼容性测试和真实 ASR 厂商联调。
9. 前端构建存在 chunk size 警告，暂不影响当前原型运行。
10. 开发环境通过 Vite 代理访问后端；后端当前未配置通用 CORS。
11. 本次新增测试覆盖了日志、提醒、设备别名的跨用户隔离，但尚未扩展到所有业务实体的多用户/多家庭隔离专项测试。
